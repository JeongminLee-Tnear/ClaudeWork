## gstack

웹 브라우징이 필요할 때는 반드시 /browse 스킬(gstack)을 사용할 것. `mcp__claude-in-chrome__*` 도구는 사용 금지.

사용 가능한 gstack 스킬:
- /office-hours — 브레인스토밍, 아이디어 논의
- /plan-ceo-review — CEO 관점 전략 리뷰
- /plan-eng-review — 엔지니어링 아키텍처 리뷰
- /plan-design-review — 디자인 리뷰
- /design-consultation — 디자인 컨설팅
- /review — 코드 리뷰
- /ship — 출시 준비
- /land-and-deploy — 머지 및 배포
- /canary — 카나리 배포
- /benchmark — 벤치마크
- /browse — 웹 브라우징 (모든 웹 브라우징에 이것을 사용)
- /qa — QA 테스트
- /qa-only — QA만 실행
- /design-review — 비주얼 디자인 리뷰
- /setup-browser-cookies — 브라우저 쿠키 설정
- /setup-deploy — 배포 설정
- /retro — 회고
- /investigate — 디버깅/조사
- /document-release — 릴리즈 문서화
- /codex — 세컨드 오피니언
- /cso — CSO
- /autoplan — 자동 리뷰
- /careful — 프로덕션 안전
- /freeze — 스코프 제한 편집
- /guard — 프로덕션 안전
- /unfreeze — 스코프 제한 해제
- /gstack-upgrade — gstack 업그레이드
